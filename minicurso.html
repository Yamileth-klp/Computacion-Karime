Sesión 1: Fundamentos de Python

¿Qué es Python?

Python es uno de los lenguajes de programación más utilizados en el mundo. Se usa para crear aplicaciones, videojuegos, páginas web, inteligencia artificial y análisis de datos.

Mostrar información

print("Hola Mundo")

Variables

nombre = "Yamileth"
edad = 17

print(nombre)
print(edad)

Entrada de datos

nombre = input("¿Cómo te llamas? ")

print("Hola", nombre)

Actividad

Crea un programa que pregunte tu nombre, edad y ciudad. Después muestra toda la información en pantalla.

Sesión 2: Condicionales y Toma de Decisiones

¿Qué es un if?

Permite que el programa tome decisiones dependiendo de una condición. 

edad = 18

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

Comparaciones

  • == Igual
  • != Diferente
  • > Mayor que
  • < Menor que
  • >= Mayor o igual
  • <= Menor o igual

Ejemplo práctico

calificacion = 90

if calificacion >= 70:
    print("Aprobado")
else:
    print("Reprobado")

Actividad

Crear un programa que indique si un alumno aprobó o reprobó.

Sesión 3: Ciclos, Listas y Proyecto Final

Listas

frutas = ["Manzana", "Pera", "Uva"]

print(frutas)

Ciclo for

for fruta in frutas:
    print(fruta)

Funciones

def saludar(nombre):
    print("Hola", nombre)

saludar("Yamileth")

Proyecto Final

Crear un sistema que registre nombres de alumnos y los muestre al final. 

alumnos = []

for i in range(3):
    nombre = input("Nombre: ")
    alumnos.append(nombre)

print("Lista de alumnos:")

for alumno in alumnos:
    print(alumno)
