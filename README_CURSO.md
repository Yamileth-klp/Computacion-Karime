# 🐍 Mini Curso de Python Básico

## Descripción

Este proyecto consiste en un mini curso interactivo de Python Básico desarrollado en HTML, CSS y JavaScript. El curso está diseñado para personas que desean comenzar a programar desde cero y aprender los conceptos fundamentales de Python de manera sencilla y práctica.

A través de tres sesiones, el estudiante conocerá los temas más importantes para iniciar en la programación, siguiendo una secuencia de aprendizaje progresiva que facilita la comprensión de los conceptos básicos.

## Objetivo

Proporcionar una introducción clara y práctica al lenguaje de programación Python, permitiendo que los estudiantes desarrollen habilidades básicas para crear programas sencillos y comprender la lógica de programación.

## Contenido del Curso

### Sesión 1: Fundamentos de Python

* Introducción a Python
* Función `print()`
* Variables
* Tipos de datos
* Entrada de datos con `input()`
* Actividad práctica

### Sesión 2: Condicionales y Toma de Decisiones

* Operadores de comparación
* Estructuras `if`, `elif` y `else`
* Evaluación de condiciones
* Actividades prácticas

### Sesión 3: Ciclos, Listas y Funciones

* Listas
* Ciclos `for`
* Funciones básicas
* Proyecto final de registro de alumnos

## Tecnologías Utilizadas

* HTML5
* CSS3
* JavaScript

## Características

* Navegación entre sesiones mediante botones.
* Diseño simple e intuitivo.
* Ejemplos de código Python.
* Actividades prácticas para reforzar el aprendizaje.
* Proyecto final para aplicar los conocimientos adquiridos.

## Resultados de Aprendizaje

Al finalizar el curso, el estudiante será capaz de:

* Comprender los conceptos básicos de Python.
* Utilizar variables y tipos de datos.
* Solicitar información al usuario.
* Implementar estructuras condicionales.
* Utilizar ciclos para automatizar tareas.
* Crear funciones simples.
* Desarrollar programas básicos en Python.

Proyecto educativo desarrollado como práctica para la enseñanza de los fundamentos de programación en Python.
